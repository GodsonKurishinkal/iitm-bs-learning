## üî¨ Core Course Deep Dive

### 1. MLOps (BSDA5014)
**Topics**:
- ML model lifecycle management
- CI/CD for ML
- Model monitoring and versioning
- Feature stores
- Model serving at scale
- A/B testing for ML

**Tools**: MLflow, Kubeflow, Docker, Kubernetes

### 2. Mathematical Foundations of Generative AI (BSDA5002)
**Topics**:
- Probability theory for GenAI
- Variational inference
- Diffusion models mathematics
- Transformer architecture theory
- Optimization for large models

**Prerequisite Knowledge**: Strong math background

### 3. Algorithms for Data Science (BSDA5003)
**Topics**:
- Randomized algorithms
- Approximation algorithms
- Streaming algorithms
- Sketching and sampling
- Graph algorithms for big data

**Applications**: Large-scale data processing
