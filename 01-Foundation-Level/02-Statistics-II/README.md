## ðŸ” Key Concepts Quick Reference

### Probability & Distributions
- **Joint Distribution:** P(X, Y) - probability of X and Y occurring together
- **Independence:** P(X, Y) = P(X) Ã— P(Y)
- **Covariance:** Measure of linear relationship between variables
- **Correlation:** Standardized covariance (-1 to +1)
- **CLT:** Sample means approach normal distribution as n increases

### Estimation
- **Point Estimate:** Single value estimate of parameter (e.g., xÌ„ for Î¼)
- **Confidence Interval:** Range likely to contain true parameter
- **MLE:** Maximum Likelihood Estimation - most likely parameter value
- **95% CI:** 95% of such intervals contain the true parameter

### Hypothesis Testing
- **Null Hypothesis (Hâ‚€):** Status quo, no effect
- **Alternative Hypothesis (Hâ‚):** What we're testing for
- **p-value:** P(observe data | Hâ‚€ is true)
- **Significance Level (Î±):** Threshold for rejecting Hâ‚€ (typically 0.05)
- **Type I Error:** False positive (rejecting true Hâ‚€)
- **Type II Error:** False negative (failing to reject false Hâ‚€)
- **Power:** Probability of correctly rejecting false Hâ‚€

### Regression
- **Simple Linear Regression:** Y = Î²â‚€ + Î²â‚X + Îµ
- **RÂ²:** Proportion of variance explained (0 to 1)
- **Residuals:** Observed - Predicted values

## ðŸ”¬ Statistical Thinking

> "Statistics is about reducing uncertainty, not eliminating it."

Key principles to remember:
1. **Always visualize your data first**
2. **Check assumptions before applying tests**
3. **p < 0.05 doesn't mean the effect is large or important**
4. **Correlation â‰  Causation**
5. **Focus on effect sizes, not just significance**
6. **Consider practical significance vs statistical significance**
7. **Be honest about limitations and assumptions**
