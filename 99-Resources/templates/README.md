# Quick Start Templates

This folder contains templates for consistent note-taking and project organization.

## Note Templates

### Lecture Notes Template
Use this for class notes:
```markdown
# [Topic Name]

**Date**: YYYY-MM-DD
**Week**: X
**Subject**: [Subject Name]
**Tags**: #tag1 #tag2

## Summary
Brief overview

## Key Concepts
- Concept 1
- Concept 2

## Examples
Example problems and solutions

## Questions
- [ ] Question 1

## Next Steps
- [ ] Action item 1
```

## Jupyter Notebook Structure

Every notebook should include:
1. Title cell (Markdown)
2. Setup/Imports (Code)
3. Learning sections (Markdown + Code)
4. Practice exercises (Code cells)
5. Notes section (Markdown)
